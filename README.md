# DevOps Portfolio

A project designed to showcase a DevOps portfolio of public work.

## Description

The AWS DevOps portfolio is a project designed to showcase expertise in CI/CD pipelines, cloud computing, and infrastructure management. Tracking the progress of this development is intended to both inspire LinkedIn posts/blog articles and enhance AWS skills. By building and deploying this DevOps portfolio, potential employers and collaborators will be able to assess and review skills otherwise only displayed in proprietary spaces.
